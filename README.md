# 🌟 Overview
Non-Player Character for LLaMA is a groundbreaking Minecraft mod that replaces traditional scripted villager AI with dynamic, generative intelligence powered by Meta's LLaMA language models. This mod allows you to spawn NPCs that understand natural language, remember past interactions, exhibit unique personalities, and react to the world around them in real-time.

Why this mod?

🧠 True Intelligence: No more repetitive dialogue trees. Every conversation is unique.

🎭 Dynamic Personalities: Define traits like curious, grumpy, helpful, or mysterious.

🌍 World Awareness: NPCs react to time, weather, player actions, and nearby events.
